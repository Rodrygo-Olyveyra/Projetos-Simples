FAQs

O que é o Frontend Mentor e como ele me ajudará?

O Frontend Mentor oferece desafios de programação realistas para ajudar desenvolvedores a aprimorar suas habilidades de programação frontend com projetos em HTML, CSS e JavaScript. É adequado para todos os níveis e ideal para a construção de portfólios.

O Frontend Mentor é gratuito?

Sim, o Frontend Mentor oferece desafios de programação gratuitos e premium...

Posso usar projetos do Frontend Mentor no meu portfólio?

Sim, você pode usar projetos concluídos no Frontend Mentor no seu portfólio...

Como posso obter ajuda se estiver preso em um desafio do Frontend Mentor?

O melhor lugar para obter ajuda é na comunidade do Discord do Frontend Mentor...
